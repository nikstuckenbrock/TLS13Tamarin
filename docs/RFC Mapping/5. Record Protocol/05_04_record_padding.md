# [5.4. Record Padding](https://datatracker.ietf.org/doc/html/rfc8446#section-5.4)

/// html | div[style='float: left; width: 50%; padding: 5px;']
```title="RFC 8446"
All encrypted TLS records can be padded to inflate the size of the
TLSCiphertext.  This allows the sender to hide the size of the
traffic from an observer.
```
